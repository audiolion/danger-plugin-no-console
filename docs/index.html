    danger-plugin-no-console

    Danger plugin to prevent merging code that still has console.logs inside it.

    Usage

    Install:

    yarn add danger-plugin-no-console --dev

    At a glance:

    // dangerfile.js
import noConsole from 'danger-plugin-no-console'

noConsole()
